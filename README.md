The game is called dark-chess, which is a chess variant that you can also find it on chess.com(This is from where the inspiration came).

Features:

  - Fog of war: You can see boxes based on your pieces position(like if u have the queen in the middle thats the maximum visibility you can get)
  - Win condition: The game ends when one sides captures the king of the other
  - Highlights: When you select a piece the box becomes green

Installation & requirements:

  - Visual Studio 2022 or above
  - Clone the repository in Visual Studio 2022 then build it then you can run the game.

How to play:

  - Just use your left click to select and deselect pieces
  - Need one more person to play with 

Tech Stack:

  - Language: C#
  - Framework: Winforms
